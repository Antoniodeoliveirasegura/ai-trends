Mastra AI Framework Hit by npm Supply-Chain Attack — 144 Packages, 1.1M Weekly Downloads Exposed

On June 17 an attacker hijacked ehindero, a real former contributor's npm account whose scope access was never revoked, and silently republished 141 packages in the @mastra namespace between 01:12–02:36 UTC. Each poisoned package added easy-day-js — a typosquat of dayjs — as a dependency; its postinstall hook downloaded and executed a remote-access trojan, then self-deleted. The blast radius: @mastra/core alone has ~918k weekly downloads, and combined affected packages exceed 1.1 million installs per week.

This matters because Mastra is the dominant JavaScript framework for building AI agents — CI pipelines, LLM tool-chains, and production deployments worldwide run it. Socket flagged easy-day-js within six minutes and blocked installs for protected users; npm has since reverted the highest-profile packages, but smaller scoped packages and the dropper itself can still resolve to the payload. Any team that ran npm install between roughly 01:00–08:00 UTC today should treat the host as compromised.
